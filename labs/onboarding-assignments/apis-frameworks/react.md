# React

React uses either Javascript or Typescript. If you don't know one of those already, complete that assignment first:

[Javascript](../Programming%20Languages/Javascript%20ff2b975d275f42feb64a11df6dcb21be.md)

[TypeScript ](../Programming%20Languages/TypeScript%20c151672487004ccc91c99e3260e44981.md)

Complete and submit any one of the following:

- Resulting code (in a Codepen or Github) from the official React tutorial [https://reactjs.org/tutorial/tutorial.html](https://reactjs.org/tutorial/tutorial.html) — this might be a better fit if your project is to create a website
- Resulting code (in a Codepen or Github) from the [Expo.io](http://expo.io) tutorial [https://docs.expo.io/get-started/installation/](https://docs.expo.io/get-started/installation/) — this might be a better fit if your project is specifically to make a mobile app
