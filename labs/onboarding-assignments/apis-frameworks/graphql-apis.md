# GraphQL APIs

Follow along with this course on creating GraphQL queries: [https://www.youtube.com/watch?v=omSpI1Nu_pg](https://www.youtube.com/watch?v=omSpI1Nu_pg)

Using the demo GraphQL API available in the link of the video, write a query that displays the average height and lifespan of all species in the dataset, as well as the total count of species. Use a fragment for the height and lifespan.

Upload a screenshot of your query with results to [Imgur.com](http://imgur.com) and submit the result to complete this assignment.
