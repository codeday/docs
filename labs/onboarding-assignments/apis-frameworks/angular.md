# Angular

Angular uses Javascript or Typescript. If you're not already familiar, complete one of those two lessons first:

[Javascript](../programming-languages/javascript.md)

[TypeScript](../programming-languages/typescript.md)

Complete **one** of the following options and submit your code (Github/Repl) to complete the assignment:

- The official Angular tutorial — [https://angular.io/start](https://angular.io/start) — through the end of "Getting Started → Try It → Deploying an application"
- This video course — [https://www.youtube.com/watch?v=k5E2AVpwsko](https://www.youtube.com/watch?v=k5E2AVpwsko) (you do not need to purchase the full course, just stop at the end of the video and upload what you've created up to that point)
