# Containers (Docker)

Complete and submit any one of the following:

- Complete this tutorial: [https://stackify.com/docker-tutorial/](https://stackify.com/docker-tutorial/)

Upload your Dockerfile to Github or Pastebin and submit the link.
- Complete this tutorial *only through "Webapps with Docker → Dockerfile*: [https://docker-curriculum.com/](https://docker-curriculum.com/#dockerfile) (may be most suitable if you're in the advanced track)

Upload your Dockerfile to Github or Pastebin and submit the link.
- Watch this video([https://fireship.io/lessons/docker-basics-tutorial-nodejs/](https://fireship.io/lessons/docker-basics-tutorial-nodejs/)) and follow along to complete this tutorial. Once done, add your own twist to it to prove what you learned.
    
    
    Upload your Dockerfile to Github or Pastebin and submit the link.
    

- Watch the CodeDay Labs talk from 2020 about Docker and other cloud deployment technologies: [https://www.youtube.com/watch?v=cy-DrN4r3ME](https://www.youtube.com/watch?v=cy-DrN4r3ME&t=2839s) (may be most suitable if you're in the advanced track)

Create and run a simple Dockerfile to run a program written in a language of your choice, and submit it (Github or Pastebin link).
