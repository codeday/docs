# WebGL

Many projects which are using WebGL are actually using another framework, which will be specified in the project description. Check your project description to figure out which assignment you should complete.

- **Three.js:**
Watch the following video [https://fireship.io/lessons/threejs-tutorial-portfolio/](https://fireship.io/lessons/threejs-tutorial-portfolio/)

Push your work to a Github repository and submit it.

{: .note-title}
> 🎓 **What if I already know WebGL?**
> 
> Please pick assignments for aspects of your tech stack which you don't already know.
> 
> If you can demonstrate that you already know all aspects of your tech stack, please contact us on Slack.


{: .note-title}
> ❓ **What if I need help with WebGL?**
> 
> 
> You can ask for help in #ask-for-help or schedule a CSE session.
