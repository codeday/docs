# Javascript

{: .note}
> 👉 Only choose the option which best aligns with how you learn! You don't need to do all of them!


Complete and submit any **one** of the following:

- A screenshot (upload it to [Imgur.com](http://imgur.com) to get a URL) of your progress through the end of "Classes" on [https://www.codecademy.com/courses/introduction-to-javascript/lessons/introduction-to-javascript/exercises/intro](https://www.codecademy.com/courses/introduction-to-javascript/lessons/introduction-to-javascript/exercises/intro)

* May require a Pro subscription. Codecademy has a free 7 day trial. If you need a burner credit card to trial with, please let us know.
- A screenshot (upload it to Imgur to get a URL) of your progress through section "6. Arrays I" on [https://learnjavascript.online/](https://learnjavascript.online/app.html?id=1437)
- Follow along with this Javascript tutorial video: [https://www.youtube.com/watch?v=W6NZfCO5SIk](https://www.youtube.com/watch?v=W6NZfCO5SIk)

Once you understand the basics, create a new Javascript project in [Repl.it](http://repl.it): [https://replit.com/new/nodejs](https://replit.com/new/nodejs)

Create a program which does the following, then save and submit the link to your Repl.it project:
    - For each number from 1-100, the program prints the number (using console.log)
    - If the number is evenly divisible by 3, the program also prints "fizz"
    - If the number is evenly divisible by 5, the program also prints "buzz"
    - If it's evenly divisible by both, it prints "fizzbuzz"
    - The program should use a method to determine what to print.
