# Java

> 👉 Only choose the option which best aligns with how you learn! You don't need to do all of them!


Complete and submit any **one** of the following:

- A screenshot (upload it to [Imgur.com](http://imgur.com) to get a URL) of your progress through the end of "Debugging" on [https://www.codecademy.com/courses/learn-java/lessons/hello-world-java/exercises/introduction-to-java](https://www.codecademy.com/courses/learn-java/lessons/hello-world-java/exercises/introduction-to-java)

- Complete the entirety of the "Learn the Basics" series here: [https://www.learnjavaonline.org/](https://www.learnjavaonline.org/)

Once you understand the basics, create a new Python project in [Repl.it](http://repl.it): [https://replit.com/new/java10](https://replit.com/new/java10)

Create a program which does the following, and then save and submit the URL of your Repl.it project:
    - For each number from 1-100, the program prints the number (using console.log)
    - If the number is evenly divisible by 3, the program also prints "fizz"
    - If the number is evenly divisible by 5, the program also prints "buzz"
    - If it's evenly divisible by both, it prints "fizzbuzz"
    - The program should use a method to determine what to print.
- Follow along with this Java tutorial video: [https://www.youtube.com/watch?v=eIrMbAQSU34](https://www.youtube.com/watch?v=eIrMbAQSU34)
