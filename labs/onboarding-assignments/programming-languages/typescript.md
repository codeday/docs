# TypeScript

{: .note}
> 👉 Only choose the option which best aligns with how you learn! You don't need to do all of them!


Complete and submit any **one** of the following:

- A screenshot (upload it to [Imgur.com](http://imgur.com) to get a URL) of your progress through the end of "Advanced Object Types" on [https://www.codecademy.com/courses/learn-typescript/lessons/introduction-to-typescript/exercises/from-javascript-to-typescript](https://www.codecademy.com/courses/learn-typescript/lessons/introduction-to-typescript/exercises/from-javascript-to-typescript) (ONLY recommended if you already know Javascript)

* May require a Pro subscription. Codecademy has a free 7 day trial. If you need a burner credit card to trial with, please let us know.
- Follow along with this Typescript tutorial video: [https://www.youtube.com/watch?v=gp5H0Vw39yw](https://www.youtube.com/watch?v=gp5H0Vw39yw)

Once you understand the basics, create a new Typescript project in [Repl.it](http://repl.it): [https://replit.com/new/typescript](https://replit.com/new/typescript)

Create a program which does the following, and then save and submit the URL of your Repl.it project:
    - For each number from 1-100, the program prints the number
    - If the number is evenly divisible by 3, the program also prints "fizz"
    - If the number is evenly divisible by 5, the program also prints "buzz"
    - If it's evenly divisible by both, it prints "fizzbuzz"
    - The program should use a method to determine what to print.
